Se refiere a la validación del apoyo educativo cuando se determina una decisión. Se cuenta con la opción de asociar la combinación del apoyo educativo con el catálogo dado de alta en Salesforce. <br/><br/>Reglas de negocio:<br/><ul>
<li>Mostrar opciones de las combinaciones de beca que se pueden realizar,derivadas del Calculo para Complementar la beca con el % de crédito de forma automatica.</li><li>De acuerdo a las combinaciones configuradas en B9.4 Combinaciones posibles entre becas, descuentos y prestaciones.</li><li>Hasta 25% de crédito</li><li>Máximo apoyo de Beca+Crédito= 80</li><li>Se debe generar el resultado integrando la beca otorgada capturada y el complemento de crédito, mostrando ambos datos.</li><li>Si se se aceptan las combinaciones, entonces:</li></ul>
<ol>
<li>Guardar Beca Actual y Beca con Combinaciones (Se le mostrarán al candidato en el proceso de aceptación de beca)</li></ol>
<ul>
<li>No se aceptan las combinaciones, entonces:</li></ul>
1. Mostrar opción para Seleccionar modificar combinaciones propuestas.<br/>2. Mostrar opción para capturar las combinaciones de beca que se pueden realizar y Permitir actualizar los porcentajes de la beca otorgada y el préstamo, respetando las validaciones<br/>
